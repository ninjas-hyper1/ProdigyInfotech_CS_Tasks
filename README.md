Cyber Security Internship - Prodigy InfoTech

This repository contains the work I completed during my **Cyber Security Internship at Prodigy InfoTech**.  
The main goal of this internship was to understand basic cybersecurity concepts and apply them through small practical tasks.

I have tried to keep each project simple and focused so that the concepts are easy to understand.

---

## Projects

### 1. Caesar Cipher
In this task, I implemented a basic encryption and decryption system using the Caesar Cipher technique.  
It helped me understand how classical encryption works.

- Language: Python  
- Concepts: Cryptography, string handling  

---

### 2. Image Pixel Manipulation
This project is based on hiding and extracting messages inside images.  
It gave me a basic idea of how steganography works.

- Language: Python  
- Library: PIL  
- Concepts: Image processing, data hiding  

---

### 3. Password Strength Checker
In this task, I created a tool that checks how strong a password is based on different conditions like length and characters.  
It also suggests improvements.

- Language: Python  
- Concepts: Password security, validation  

---

### 4. Keylogger (Educational Purpose Only)
This project demonstrates how a keylogger works.  
It records keystrokes and stores them in a file. This was done only to understand security risks and prevention.

- Language: Python  
- Concepts: Ethical hacking basics, security awareness  

---

### 5. Network Packet Analyzer
This task helped me understand how network packets are captured and analyzed.  
It is useful for monitoring traffic and identifying suspicious activity.

- Language: Python  
- Library: Scapy  
- Concepts: Networking, packet analysis
